# Motorsport Telemetry Analyzer

This project is a telemetry analysis tool developed for my thesis.

Features:
- CSV telemetry upload
- Speed visualization
- Lap time calculation
- Track visualization
- Lap comparison

Technologies:
- Java
- Flask
- Chart.js
- Java (desktop application later)
